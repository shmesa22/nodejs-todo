<h1>README</h1>

<p>1. Clone this repository: </p>
<pre>
  <code>
    git clone https://github.com/shmesa22/nodejs-todo
  </code>
</pre>
<p>2. Navigate to the directory: </p>
<pre>
  <code>
    cd nodejs-todo
  </code>
</pre>
<p>3. run mongoDB on the console: </p>
<pre>
  <code>
    mongod --port 27017 --dbpath ./db
  </code>
</pre>
<p>4. Start the server: </p>
<pre>
  <code>
    npm run start
  </code>
</pre>
